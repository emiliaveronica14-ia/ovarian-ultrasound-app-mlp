# ovarian-ultrasound-app-mlp
Physiological &amp; Pathological Ovarian Conditions Classifier 🩺🥼 Upload an ultrasound image to receive an automated predictive diagnosis based on a custom MLP architecture. There are 5 possibilities for prediction of ovarian conditions:. 1️⃣ Healthy Ovary, 2️⃣ Polycystic Ovary (PCOS), 3️⃣ Simple Cyst, 4️⃣ Complex Cyst, and 5️⃣Dominant Follicle.
